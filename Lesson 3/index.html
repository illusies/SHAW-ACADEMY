What is html?

Hypertext Markup Language (HTMl) is the standard language for creating web pages and web application. Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page sematifically and originally included cues for the appearance of the document.

Characteristics of HTML

  1. Uses elements/tags to build web page structures
  2. Enables media files to be embedded in webpages i.e. audio and video clips
  3. Semantically defines web page for text i.e. paragraphs, headings etc.
  4. Allows developers to embed JavaScript code
  5. Elements/tags have attributes
